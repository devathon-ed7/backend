## Getting started

1. clone this repository

   ```sh
   git clone https://github.com/kacubillos/devathon-backend
   ```

2. install dependencies

   ```bash
   yarn install
   ```

3. create .env file

   ```bash
   touch .env
   ```

4. write to .env

   ```bash
   DATABASE_URL = "mysql://janedoe:mypassword@localhost:3306/mydb"
   ```

5. run the project
   ```bash
   yarn run dev
   ```
